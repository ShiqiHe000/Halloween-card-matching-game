# Halloween-card-matching-game
A Halloween theme card matching game by HTML, CSS and JavaScript. 

## Practise
* CSS grid. 
* Customized fonts. 
* Customized cursor. 
* CSS animations. 
* CSS transform and transition. 
* JavaScript audio. 
* OOP programming. 
* Game strategy planning. 

## How to play? 
Open the `index.html` file in the browser.
